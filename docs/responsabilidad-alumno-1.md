# Responsabilidad alumno 1

## Alcance

Esta rama representa el trabajo de frontend y catalogo visual del proyecto Milwaukee Bucks.

## Tareas principales

- Estructura de navegacion del catalogo.
- Home principal del portal.
- Vistas informativas: snapshot, roster, calendario e identidad.
- Componentes reutilizables de presentacion.
- Estilos responsive y tema visual Bucks.
- Interactividad de filtros del roster y cambio de tema.

## Relacion con el TP 2

Esta rama cubre la base visual sobre la cual se integra la funcionalidad serverless.
La rama `alumno-2` contiene la capa de autenticacion, CRUD y persistencia en Supabase.
